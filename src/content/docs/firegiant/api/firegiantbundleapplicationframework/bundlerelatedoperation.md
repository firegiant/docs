Relation types between packages.
## Members
| Member | Description |
| ------ | ----------- |
| Downgrade | The related package will be downgraded. |
| Install | The related package will be installed. |
| MajorUpgrade | The related package will be upgraded as a major revision. |
| MinorUpgrade | The related package will be upgraded as a minor revision. |
| None | No related operation. |
| Remove | The related package will be removed. |
| Repair | The related package will be repaired. |

`FireGiant.BundleApplicationFramework v6.0.1`
