Package state.
## Members
| Member | Description |
| ------ | ----------- |
| Absent | Package is not installed. |
| Cached | No longer used, see the BundlePackage.Cached property instead. |
| Obsolete | Package is obsolete. |
| Present | Package is installed. |
| Superseded | Package is superseded by another package. |
| Unknown | Package state is unkown. |

`FireGiant.BundleApplicationFramework v6.0.1`
