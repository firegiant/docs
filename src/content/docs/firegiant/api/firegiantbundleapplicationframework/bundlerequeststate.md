Request state.
## Members
| Member | Description |
| ------ | ----------- |
| Absent | Uninstall and remove from cache. |
| Cache | Cache if not installed, uninstall if installed. |
| ForceAbsent | Force removal of bundle or package, including permanent packages. |
| None | No request state or no change to request state. |
| Present | Install if not present. |
| Repair | Repair if installed, install if not installed. |

`FireGiant.BundleApplicationFramework v6.0.1`
